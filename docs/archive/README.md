# Atlas Documentation Archive

This folder contains documents that are useful historical context but should not be treated as current implementation guidance.

Use the active docs in `docs/README.md` for current product, architecture, and MVP execution decisions.

## Archive Groups

- `architecture/` - Superseded architecture and frontend design drafts.
- `conversations/raw/` - Raw conversation exports and intermediate analysis.
- `handoffs/` - Handoff notes for reverted or paused implementation work.
- `previews/` - Standalone HTML previews and visual artifacts.
- `product/` - Product snapshots or research notes superseded by active product docs.
- `reviews/` - Point-in-time reviews and readiness snapshots.

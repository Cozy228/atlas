# Platform & Developer Experience (DevEx)

## Key Questions Raised

- Is there a centralized portal for managing landing zones?
- How do teams navigate documentation and platform capabilities?

## Current State

- Only the central landing zone has a custom portal.
- Other environments use:
  - Terraform Enterprise
  - Harness CI/CD
- No unified developer portal exists.

## Challenges Identified

- Documentation is fragmented and outdated.
- Discoverability is limited across tools and environments.

## Future Direction

- Consolidate knowledge portals.
- Consider a DevEx portal that is information-centric, not provisioning-centric.
- Explore AI-based documentation discovery and maintenance.

## Implication

- Platform maturity depends on experience, not just tooling.
- There is a clear need for a "Platform as a Product" UX layer.
